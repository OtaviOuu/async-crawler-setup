Bem, aqui primeiro vamos fazer uma manipulação para encontrar o valor dessa integral:

∫ 3 x + 1 d x = ∫ 3 x d x + ∫ 1 d x

Agora fica simples de resolver, vamos lá, usando duas expressões da Tabela de Integrais Indefinidas:

∫ x n d x = x n + 1 n + 1 + C   n ≠ - 1

∫ k   d x = k x + C

Agora é só aplicar e pronto:

∫ 3 x d x + ∫ 1 d x = 3 ∫ x d x + x + C = 3 ⋅ x 1 + 1 1 + 1 + x + C = 3 x 2 2 + x + C
